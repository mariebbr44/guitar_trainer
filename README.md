# Guitar Trainer
Application to aid in the guitar training process

## Running the Application

### Command Line
To run the application on the command line, python3 and tkinter must be installed. Then, while in the project home directory, run the command ```python3 guitar_trainer.py```. The application will appear in a separate window.

### Standalone Mac Application
To convert the program into a standalone mac application, first install py2app, then run the command ```python3 setup.py py2app``` while in the project home directory.
